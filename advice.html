Advice

The Green Carbon Calculator measures the carbon footprint of a given website and showcases the results in a pleasant and appealing manner.

About the project

Github Repository

In this project, together with my teammates, we had to create a prototype for an online calculation tool for web CO2 emission, that would stand out and would make the visitors take act on. The requirements from Advice were to create a tool that was primarily aimed for managers and other decision makers to help create awareness and facilitate dialogue about the topic.

My role in this project was to implement the design of the Figma Prototype using SASS.


Having in mind our target audience, we decided to go for a dashboard design, that shows both parameters and results in the same page, because in this case the user will not have to scroll too much in order to see the results.

Carrying out this project was an opportunity to use different technologies and tools such as: CRUD methods, REST API & Cleaning data, form manipulation, SASS, JavaScript.

Our solution has different animations and microinteractions that are meant to provide a beautiful, informative and interactive experience on the website.
Due to the fact that we are fetching data from two websites and this might take some time, we decided to add a loading animation with some text that containes practical information about how website pollution works.
In the results page we have added radio buttons and a slider that control the values of the parameters and live prints the improvements.
