Shilla's Basket

In this assignment, I have created a mini game using HTML, CSS and Vanilla Javascript, where the user needs to collect good apples in order to win the game.

About the project

Github Repository

This is a first semester project, where we were introduced to JavaScript, CSS animations and activity diagrams. For this project, we had to create all visual elements, using Adobe Illustrator, which in the beginning it was a struggle.

I have created the game, starting from my love for elephants, and this helped me choose the design style of the whole game - organic.

The game is about a baby elephant called Shilla and her love for apples. In order to win, the user needs to collect 20 apples in 30 seconds in order to fill the basket and feed Shilla.

For the UI elements of my design I went with the wood design, in order to create balance and harmony with the green background.

In the first page, the user has two options, to play the game or to learn how to play the game. If the users want to see the rules of the game, he clicks on the “How to play” button and then a new page will appear. After he reads the story, he can start playing by pressing the “Start” button from the page.

After the game starts, the user needs to click on the good apples in order to fill the basket and gain points. If the user clicks on bad apples, he loses one life, out of three. The game does not stop when the user clicks on a bad apple.
